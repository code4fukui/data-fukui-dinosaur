# data-fukui-dinosaur

**福井県恐竜博物館**（勝山市）を訪問した際に撮影された70枚の写真（2015年12月にiPhone 6で撮影）を収録したデータセットです。同博物館は世界三大恐竜博物館の一つに数えられ、日本を代表する古生物学研究機関です。

## データセットの特徴

- **建築・敷地:** 博物館を象徴する銀色のジオデシックドーム、屋外の恐竜モニュメント、近隣の勝山ジオパークにある全長17.5mの巨大なホワイトサウルス（ティラノサウルス・レックス）像。
- **恐竜展示:** 巨大な**カマラサウルスの産状化石**、実物大で動くティラノサウルス・レックスやステゴサウルスの模型、そして福井県で発見された種（*フクイサウルス*、*フクイラプトル*、*コシサウルス*、*フクイティタン*）の専用展示。
- **地質学・地球の歴史に関する展示:** 希少な鉱物標本、先カンブリア時代のストロマトライト、古生代の化石などの包括的なコレクション。
- **来館体験:** 3階分を下るらせんエスカレーター、アニメキャラクターの写真撮影用パネル、Dino Cafeの恐竜をテーマにしたメニュー（「恐竜の巣カレー」など）、記念メダルなどの限定土産。

## データセットの構成

本データセットは、静的な画像コレクションで構成されています。

- `photos/2015-12/`: 全70枚の `.jpg` 画像ファイルを収録しています。

メタデータファイルやAPIは提供されていません。

## 博物館について

福井県恐竜博物館は2000年に開館しました。地元・勝山市にある北谷層からは数多くの重要な恐竜化石が発掘されており、福井県固有の複数の新種の発見につながっています。
